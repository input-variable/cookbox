COOKBOX lights recipes

Welcome to Cookbox.

Cookbox hosts recipes in the public domain. If you would like to submit a recipe then please leave a pull request in the github repo.

If you would like to see recipes then you can press that niftly little link on the top right. Next to that is the lightswitch. It creates a small cookie in your browser so you don't have to toggle it every time you load a new page.
